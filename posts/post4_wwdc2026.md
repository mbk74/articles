# Adaptive UI should not hide new features

There is one risk with adaptive interfaces that I rarely see discussed.

If AI only shows what users already use, how will they discover something new?

A good adaptive UI should reduce complexity.

But it should also preserve discovery.

Maybe AI should decide:

* what stays visible
* what moves into "More"
* which new features deserve temporary promotion

The goal isn't to hide functionality.

The goal is to manage attention.
