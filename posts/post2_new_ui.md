# AI complexity for UI

For years, UX was built around navigation.
More screens/menus/filters/settings.
AI gives us a chance to rethink this.
I don’t think screens will disappear.
But I think overloaded interfaces might.

One of the biggest problems with current AI systems is that users still need to "work for the AI":
write detailed prompts, explain context, carefully formulate requests, clarify intent multiple times.

Sometimes explaining the task takes longer than doing it manually.

That’s not good UX.

A good interface should reduce cognitive load, not shift it from buttons to prompts.

The future UI may become simpler on the surface while AI handles complexity quietly in the background.

Not replacing interfaces - but removing unnecessary friction.
