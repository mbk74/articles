# Adaptive UI

Adaptive UI sounds great until it starts hiding things.

Imagine an application that learns from your behavior and only shows the features you use most often.

The interface becomes cleaner.

But there is a catch.

How do users discover new functionality?

I think future AI-powered interfaces will need to balance two competing goals:

Reduce complexity.

Preserve discoverability.

The challenge is not deciding what to hide.

The challenge is deciding what deserves attention right now.
